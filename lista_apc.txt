Lista de Exercícios – Algoritmos e Programação de Computadores

Tópico 1: Estruturas de Repetição e Cálculo
1. Analise o código abaixo e explique o que ele faz. Qual valor será impresso?
   resultado = 1
   for i in range(1, 6):
       resultado *= i
   print(resultado)

2. O que aconteceria se o intervalo do for fosse alterado para range(1, 7)?

Tópico 2: Sequências e Lógica de Programação
3. Analise o código a seguir e responda:
   anterior = 0
   atual = 1
   n = int(input())
   while anterior <= n:
       print(anterior, end=" ")
       proximo = anterior + atual
       anterior = atual
       atual = proximo
   a) Para n = 20, quais valores são exibidos?
   b) O que muda ao inverter as duas últimas linhas do laço?

Tópico 3: Validação de Dados
4. Considere o código:
   soma = 0
   for i in range(1, 6):
       nota = float(input())
       while nota < 0 or nota > 10:
           nota = float(input())
       soma += nota
   media = soma / 5
   print(media)
   a) Quais valores de entrada seriam desconsiderados?
   b) Qual é a média final se as entradas forem: 7.0, -1.0, 8.5, 9.0, 11.0, 6.0, 5.5?

Tópico 4: Estruturas de Controle
5. Analise o código:
   numero = int(input('Digite um numero entre 1 e 5: '))
   while numero < 1 and numero > 5:
       print('Fora do intervalo!')
       numero = int(input('Digite novamente: '))
   print('Numero valido:', numero)
   a) O código funciona corretamente? Justifique.
   b) Corrija o erro, se houver.
