Lista de Exercícios – Arquitetura MVC

Parte 1: Teoria

Tópico 1: Conceitos Gerais de MVC

1. Explique o que é o padrão MVC e quais são as responsabilidades de cada camada.
2. Dê exemplos de situações em que a separação entre Model, View e Controller pode melhorar a manutenção de um sistema.

Tópico 2: Responsabilidades das Camadas
3. Quais tipos de lógica devem ser implementados na camada Model? Dê exemplos.
4. Por que a View não deve acessar diretamente o banco de dados? Explique os riscos dessa prática.
5. O que caracteriza uma responsabilidade do Controller? Cite exemplos de boas práticas.

Tópico 3: Boas Práticas de Arquitetura
6. Por que é importante evitar acoplamento direto entre View e Controller? Como isso pode ser feito?
7. Explique a importância de validar dados tanto na View quanto no Model. Quais problemas podem ocorrer se a validação for feita apenas na interface?
8. Descreva como a lógica de negócio deve ser separada da lógica de apresentação em um sistema bem estruturado.

Tópico 4: Persistência e Acesso a Dados
9. Qual camada deve ser responsável pelo acesso ao banco de dados? Justifique sua resposta.
10. Quais são os problemas de implementar comandos SQL diretamente em Controllers ou Views?

Tópico 5: Manutenção e Evolução de Sistemas
11. Como a aplicação correta do padrão MVC facilita a manutenção e evolução de sistemas?
12. Dê exemplos de más práticas comuns em projetos que não seguem o padrão MVC e explique como evitá-las.

Tópico 6: Exemplos e Análise de Situações
13. Analise um cenário em que a lógica de desconto de vendas está no Controller. Explique onde ela deveria estar e por quê.
14. Em um sistema onde a View realiza cálculos e persistência, quais problemas podem surgir?
15. Proponha uma solução para garantir que a integridade dos dados seja mantida em todas as camadas do sistema.

// View acessando banco de dados diretamente (identifique o problema)
public class ProdutoView {
	public void exibirProdutos() {
		List<Produto> produtos = BancoDeDados.executarQuery("SELECT * FROM produtos");
		for (Produto p : produtos) {
			System.out.println(p.getNome());
		}
	}
}

		}
	}
}
